这版加了什么 🌱

  • 任务状态（5 态）：未开始 ○ / 进行中 ▶ / 暂停 ⏸ / 完成 ✓ / 遗弃 ✕。在清单里手动改（图④点状态 ⌄ 选），列表左侧图标一眼看出状态，完成/遗弃沉到「归档」并划线淡化。
  • 详情标题下一行（图②）= 可点编辑的 状态 · P0 · 截止 三个小标，点 ⌄ 就地改，不占大块。
  • 每条清单有序列号 #1 #2…（列表 + 详情都显示），方便引用、对号。
  • 开始这一格（图③）= 跳聊天页。任务名作为一个可删 chip「写周报 #1 ×」塞进输入框，点 × 删掉即可手动输入；时长沿用真实 app 的分钟/小时切换 + 预设 + 自定义，不再是原来那排假 chip。
  • 不自动完成：到点 check-in 后只往更新日志记一笔进展（图②那条「做了 1h」带橙点=系统自动记的），状态不动——大任务做两小时≠做完，完成与否你手动定。
  • 更新日志：系统自动记的（每格 check-in）+ 你随手「＋ 记一笔」混在一条时间线上。
